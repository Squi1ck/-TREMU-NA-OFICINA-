## Como jogar

1. Permite o acesso à camara quando o browser pedir
2. Mostra a mao a camara com o gesto da letra que queres
3. Mantem o gesto 1.5 segundos ate a barra verde encher — a letra e confirmada
4. Depois de 5 letras, carrega em Confirmar
5. Tens 6 tentativas para adivinhar a palavra

### Cores
- confirmar se as letras ficarem
- Verde — letra certa no lugar certo
- Amarelo — letra existe mas no lugar errado
- Cinzento — letra nao existe na palavra


### Se não tiveres camara?
Clica no separador Teclado para jogar com o teclado normal.
Tambem podes usar o teclado fisico do computador.
